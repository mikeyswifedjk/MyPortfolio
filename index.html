MY PROJECTS

FertiliSense

An Artificial Intelligent Fertility Mobile Application using Predictive Analytics in the Reproductive System empowers users to forecast fertility windows with enhanced accuracy. It provides valuable insights into reproductive health by leveraging data analytics and personalized predictions. Users can log symptoms such as cramps, mood changes, and other physical conditions, allowing the app to track health patterns over time. The system evaluates logged symptoms and offers tailored advice for managing reproductive health effectively.

ShopBee

I developed a full-featured e-commerce website inspired by Shopee, Amazon, and Lazada. The platform includes user-friendly product listings, a secure checkout process, real-time order tracking, and customer review management. An admin panel allows control over product management, user accounts, sales reports, and promotional offers. Security measures include encrypted payments and role-based access control. The site is fully responsive, scalable, and optimized for performance, supporting seamless user experiences across all devices.

ShopSavvy

ShopSavvy is like Shopee, but I used a different approach in its development. Instead of traditional methods, I integrated AJAX for real-time updates, JSON for data handling, and jQuery for dynamic user interactions, with XML used for data storage and management. The platform features product listings, secure checkout, order tracking, and supports both an admin panel and a dedicated seller side for managing inventory, product uploads, and sales performance. Sellers can monitor orders, update stock, and communicate with customers directly through the platform. This approach ensures a smooth, scalable, and responsive experience for buyers, sellers, and administrators alike.

Solaire Hotel

I developed a full-featured hotel e-commerce platform inspired by popular booking sites like Agoda and Booking.com. The platform includes user-friendly hotel listings, secure booking options, real-time availability updates, and customer review management. An admin panel allows control over room inventory, booking management, user accounts, and promotional offers. Hotel partners can manage room details, prices, and monitor reservations through a dedicated portal.

JJTM Machine

I developed a fully functional ATM machine system with an integrated admin panel for managing transactions and user accounts. The system allows users to securely withdraw, deposit, and check balances using a simple, intuitive interface. Real-time transaction updates are provided, ensuring users can track their activity efficiently. The admin panel offers detailed oversight, enabling the management of user accounts, transaction logs, and ATM maintenance schedules. Security measures, such as encryption and two-factor authentication, are implemented to ensure the safety of user data and financial transactions.

Journal Archive

I developed a comprehensive journal archive system with five distinct user roles: Admin, Author, Reader, Reviewer, and Editor. The Admin role has full control over the platform, managing user accounts, content approval, and system settings. Authors can submit articles, manage their submissions, and track the status of their work. Reviewers can evaluate and provide feedback on submitted articles, helping maintain the quality of the content. Editors are responsible for reviewing and editing articles before they are published, ensuring content accuracy and consistency. Readers have access to published articles, can download or view them, and may engage with content through comments or ratings.
